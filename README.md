## ❓ რატომ არ არის „უბრალო აპი“ App Store-დან?

**Apple-ის წესია:** iPhone-ზე აპის დასაყენებლად საჭიროა:
- **Mac + Xcode** (უფასო, მაგრამ Mac უნდა გქონდეს), ან
