## ❓ რატომ არ არის „უბრალო აპი“ App Store-დან?

**Apple-ის წესია:** iPhone-ზე აპის დასაყენებლად საჭიროა:
- **Mac + Xcode** (უფასო, მაგრამ Mac უნდა გქონდეს), ან
- **App Store** ($99/წელი Apple Developer), ან
- **TestFlight** (ისევ Mac + $99)

Android-ის მსგავსად APK-ის ჩამოტვირთვა **iPhone-ზე შეუძლებელია** — Apple ამას აკრძალავს.

**ჩვენი გადაწყვეტა:** PWA (ვებ-აპი) — Home Screen-ზე დამატების შემდეგ **როგორც აპი** გამოიყურება და მუშაობს.
