# ავტოდაილერი + SMS

**iPhone-იდან პირდაპირ** — Mac და Xcode **არ გჭირდება**.

## 📱 iPhone-ზე გახსნა (3 ნაბიჯი)

### 1. გახსენი Safari-ში
```
https://alekoqorch.github.io/alex11/
```

### 2. დააინსტალირე Home Screen-ზე
- დააჭირე **Share** (⬆️ ღილაკი ქვემოთ)
- აირჩიე **Add to Home Screen**
- დააჭირე **Add**

### 3. გაუშვი აპი
Home Screen-ზე გამოჩნდება **„ავტოდაილერი“** — დააჭირე და გამოიყენე!

## რას აკეთებს

| ფუნქცია | აღწერა |
|--------|--------|
| 📞 ავტოდაილერი | ნომრების ჩასმა → **„დაწყება“** → რიგრიგობით ირეკავს |
| ⏱ 30 წმ | პასუხის გარეშე → გათიშე → აპში დაბრუნდი → შემდეგ ნომერზე გადავა |

## გამოყენება

### დარეკვა
1. ჩაწერე/ჩასვი ნომრები (თითო ხაზზე ერთი)
2. **„დაწყება“** → ტელეფონი გაიხსნება
3. 30 წმ-ში პასუხის გარეშე → **გათიშე ზარი**
4. **დაბრუნდი აპში** (Home Screen-ის ხატულა) → ავტომატურად შემდეგ ნომერზე

### SMS
1. მიუთითე მიმღების ნომერი
2. აირჩიე შაბლონი
3. **„გაგზავნა“** → Messages → Send
