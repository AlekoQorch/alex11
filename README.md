# ავტოდაილერი + SMS

Android აპლიკაცია, რომელიც:
- **ავტომატურად ირეკავს** ჩაწერილ/ჩასმულ ნომრებზე რიგრიგობით
- **30 წამში გათიშავს** ზარს, თუ არავინ მიპასუხა
- **SMS შაბლონებს** გთავაზობს მზა ტექსტით — აირჩიეთ და დააჭირეთ „გაგზავნა“

## ფუნქციები

| ფუნქცია | აღწერა |
|--------|--------|
| 📞 ავტოდაილერი | ნომრების სია → „დაწყება“ → თითოეულზე ავტომატური ზარი |
| ⏱ 30 წმ ტаймерი | თუ არ მიპასუხეს, ზარი ავტომატურად სრულდება |
| 💬 SMS შაბლონები | 5 მზა ტექსტი ქართულად, ერთი ღილაკით გაგზავნა |
| 📋 ნომრების ჩასმა | თითო ხაზზე ერთი, ან მძიმით/წერტილ-მძიმით გამოყოფილი |

## SMS შაბლონები

1. **ზარის მოთხოვნა** — გთხოვთ დაგვიბრუნოთ ზარი
2. **შეხსენება** — დაგვიკავშირდით დღესვე
3. **ვიზიტის დადასტურება**
4. **გადახდის შეხსენება**
5. **მადლობა**

## ინსტალაცია

### საჭიროებები
- Android Studio Hedgehog (2023.1) ან უფრო ახალი
- Android ტელეფონი API 26+ (Android 8.0+)
- JDK 17

### აწყობა

```bash
# Android Studio-ში გახსენით პროექტი
# ან ტერმინალიდან:
./gradlew assembleDebug
```

APK ფაილი: `app/build/outputs/apk/debug/app-debug.apk`

## პირველი გაშვება

1. **ნებართვები** — აპი მოითხოვს:
   - ზარის განხორციელება (`CALL_PHONE`)
   - ზარის სტატუსის წაკითხვა (`READ_PHONE_STATE`)
   - SMS გაგზავნა (`SEND_SMS`)

2. **ნაგულისხმევი დაილერი (რეკომენდებული)** — ⚙ ღილაკით დააყენეთ აპი ნაგულისხმევ დაილერად. ეს საშუალებას აძლევს აპს 30 წამში ავტომატურად გათიშოს ზარი.

3. **Accessibility სერვისი (სარეზერვო)** — თუ გათიშვა არ მუშაობს, პარამეტრებში ჩართეთ „ავტოდაილერი“ accessibility სერვისი.

## გამოყენება

### ავტომატური დარეკვა
1. ჩაწერეთ ან ჩასვით ტელეფონის ნომრები
2. დააჭირეთ **„დაწყება“**
3. აპი რიგრიგობით ირეკავს — თითო ნომერზე მაქს. 30 წამი
4. **„გაჩერება“** — ნებისმიერ დროს შეწყვეტს

### SMS გაგზავნა
1. მიუთითეთ მიმღების ნომერი (ან გამოიყენება მიმდინარე/პირველი ნომერი)
2. აირჩიეთ შაბლონი რადიო ღილაკით
3. დააჭირეთ **„გაგზავნა“**

## ტექნიკური შენიშვნები

- Android 10+ ზარის პროგრამული გათიშვა შეზღუდულია — ამიტომ საჭიროა ნაგულისხმევი დაილერის როლი ან Accessibility სერვისი
- SIM ბარათი საჭიროა ზარებისა და SMS-ისთვის
- აპი განკუთვნილია პირადი/სამუშაო გამოყენებისთვის — დაიცავით ადგილობრივი კანონმდებლობა (მასობრივი SMS/ზარები)
