# ავტოდაილერი + SMS (iOS)

iPhone-ისთვis განკუთვნილი iOS აპლიკაცია (SwiftUI):

- **ავტომატურად ირეკავს** ჩაწერილი/ჩასმული ნომრებზე რიგრიგობით
- **30 წამის ტаймерი** — თუ არ მიპასუხეს, გაჩნდება შეტყობინება გათიშვისთვის
- **5 SMS შაბლონი** მზა ტექსტით — აირჩიეთ და გაგზავნეთ

> ⚠️ **მნიშვნელოვანი:** Apple iOS-ზე არ იძლევა აპებს ზარის **ავტომატურ გათიშვის** უფლებას. 30 წამის შემდეგ ხელით უნდა გათიშოთ — აპი ავტომატურად გადავა შემდეგ ნომერზე.

---

## როგორ გავხსნა / დააყენო

### საჭიროებები
- **Mac** კომპიუტერი
- **Xcode 15+** (App Store-დან უფასო): https://developer.apple.com/xcode/
- **iPhone** (iOS 16+) USB კაბელით, ან სიმულატორი

### ნაბიჯები

1. **Xcode** გახსენით
2. **File → Open** → აირჩიეთ `AutoDialerSMS.xcodeproj`
3. iPhone მიაერთეთ Mac-ს (ან აირჩიეთ სიმულატორი მენიუში)
4. **Signing & Capabilities** → აირჩიეთ თქვენი Apple ID (Team)
5. ▶ **Run** (Cmd + R) — აპი iPhone-ზე გაიხსნება

### iPhone-ზე პირველი გაშვება (Developer Mode)

თუ Xcode-იდან პირველად აყენებ:
1. iPhone: **Settings → General → VPN & Device Management** → დაTrust-ეთ developer certificate
2. iOS 16+: **Settings → Privacy & Security → Developer Mode** → ON

## გამოყენება

### 📞 ავტომატური დარეკვა
1. ჩაწერეთ/ჩასვით ნომრები (თითო ხაზზე ერთი)
2. დააჭირეთ **„დაწყება“**
3. აპი გახსნის ტელეფონს და რიგრიგობით ირეკავს
4. 30 წამში პასუხის გარეშე → გაჩნდება შეტყობინება → **ხელით გათიშეთ** → შემდეგ ნომერზე გადავა
5. **„გაჩერება“** — ნებისმიერ დროს

### 💬 SMS
1. მიუთითეთ მიმღების ნომერი
2. აირჩიეთ შაბლონი
3. **„გაგზავნა“** → Messages აპი გაიხსნება მზა ტექსტით → დააჭირეთ Send
